# Security Audit (Trail of Bits)

Security auditing skills from Trail of Bits, the world's leading security research firm.

## Capabilities

- **Static Analysis** — CodeQL/Semgrep automated vulnerability scanning
- **Supply Chain Risk** — Audit dependencies for known vulnerabilities
- **Insecure Defaults** — Find dangerous default configurations
- **Entry Point Analysis** — Map attack surface (APIs, inputs, auth boundaries)

## Usage

Ask: "Audita a segurança do app", "Verifica vulnerabilidades", "Analisa dependências", etc.
