# React Native + Expo Agent System

This skill provides 7 specialized agents for auditing and improving React Native + Expo apps.

## Agents Available

### Tier 1 — Daily Use

- **a11y-enforcer** — Accessibility auditing (WCAG compliance, screen readers)
- **design-token-guardian** — Design system consistency (colors, spacing, typography)
- **performance-enforcer** — Performance optimization (re-renders, memory, bundle size)
- **test-generator** — Automated test generation (unit, integration, e2e)

### Tier 2 — Power Tools

- **performance-prophet** — Deep performance analysis and predictions
- **security-specialist** — Security vulnerabilities, data protection, auth flows

### Tier S — Meta

- **grand-architect** — Full architecture review and improvement recommendations

## Usage

Ask naturally: "Audita a performance do app", "Verifica segurança", "Gera testes", etc.
Read the individual agent .md files for detailed instructions.
