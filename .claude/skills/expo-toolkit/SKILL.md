# Expo Toolkit

Complete toolkit for Expo app development through App Store submission.

## Skills

- **expo-config** — Expo configuration best practices
- **expo-eas** — EAS Build and Submit
- **ios-submission** — iOS App Store submission
- **android-submission** — Google Play submission
- **fastlane** — Fastlane automation
- **revenuecat** — RevenueCat in-app purchases

## Usage

Ask: "Configura EAS build", "Prepara pra submeter na App Store", "Configura RevenueCat", etc.
